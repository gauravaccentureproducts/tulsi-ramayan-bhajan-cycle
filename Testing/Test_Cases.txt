================================================================================
TULSI-RAMAYAN BHAJAN CYCLE - TEST CASES (v2 - Tulsidas-Mukesh format)
================================================================================
Comprehensive quality-check suite for the v2 31-line AABB chaupai format.
Severity tags: [BLOCKER] = must fix before commit
                [MAJOR  ] = should fix
                [MINOR  ] = nice to fix
================================================================================


------------------------------------------------------------
GROUP TC-S : STRUCTURAL TESTS
------------------------------------------------------------

TC-S-001 [BLOCKER] Folder file count
  Scope     : per kaand
  Criterion : Folder contains only .txt files matching the naming pattern;
              count is appropriate for kaand (typically 6-10).

TC-S-002 [BLOCKER] File naming convention
  Scope     : per file
  Criterion : ^<KaandName>_Ch<N>_<EpisodeTitle>\.txt$ where N is 1+.

TC-S-003 [BLOCKER] STYLE/LYRICS markers
  Scope     : per file
  Criterion : Exactly one STYLE marker and one LYRICS marker in order.

TC-S-004 [BLOCKER] Style prompt length
  Scope     : per file
  Criterion : Style block <= 1000 characters (Suno 5.5 limit).

TC-S-005 [BLOCKER] Total verse lines = 31 (v2)
  Scope     : per file
  Criterion : Lyrics body contains exactly 31 Devanagari content-lines.

TC-S-006 [BLOCKER] Verse structure 3 + 7x4
  Scope     : per file
  Criterion : 1 sthayi (3 lines) + 7 [Verse N] blocks (4 lines each) = 31.

TC-S-007 [BLOCKER] Sthayi at top
  Scope     : per file
  Criterion : First [Chorus] block appears before any [Verse N] tag.

TC-S-008 [MAJOR  ] Mid-chapter chorus return
  Scope     : per file
  Criterion : [Chorus] returns at least once mid-song (before V4 or V5).

TC-S-009 [BLOCKER] Verse count = 7
  Scope     : per file
  Criterion : Exactly 7 [Verse N] blocks numbered 1-7.

TC-S-010 [MINOR  ] Final-chapter extra chorus
  Scope     : per file (last chapter of each kaand)
  Criterion : Additional [Chorus] block after [Verse 7] before [Outro].

TC-S-011 [BLOCKER] [Outro] tag present
  Scope     : per file
  Criterion : Lyrics block ends with [Outro] tag.


------------------------------------------------------------
GROUP TC-F : FACT VALIDATION TESTS
------------------------------------------------------------

TC-F-001 [BLOCKER] No yuga error (कलियुग)
  Criterion : Word "कलियुग" does NOT appear in the lyrics.

TC-F-002 [BLOCKER] No cross-avatar conflation (Krishna butter)
  Criterion : Krishna's butter-stealing phrases absent in Ram-context.

TC-F-003 [MAJOR  ] No सुलक्षणा as proper noun
  Criterion : "सुलक्षणा" must not appear as a queen's name.

TC-F-004 [MAJOR  ] No स्त्री-वध dilemma at Tadaka
  Criterion : Use muni-aagya frame, not stri-vadh moral dilemma.

TC-F-005 [MAJOR  ] Ahalya post-redemption
  Criterion : If Ahalya appears, she rejoins Gautama.

TC-F-006 [MAJOR  ] Sumitra from both queens
  Criterion : Both queens give kheer-portions to Sumitra.

TC-F-007 [MAJOR  ] No filler/coined words
  Criterion : Banned filler list absent.

TC-F-K1 through TC-F-K7 [MAJOR] Per-kaand episode coverage
  Criterion : All canonical Tulsidas episodes for the kaand appear at
              least once across the kaand's chapters.


------------------------------------------------------------
GROUP TC-L : LYRICIST CRAFT TESTS (v2 updated)
------------------------------------------------------------

TC-L-001 [BLOCKER] AABB rhyme per chaupai (NEW v2)
  Criterion : Each 4-pada chaupai has 2 distinct rhyme-pairs:
              padas 1-2 share a tail, padas 3-4 share a different tail.
              NOT AAAA.

TC-L-002 [MAJOR  ] Long-vowel rhyme endings
  Criterion : Each pada ends on a long vowel or nasal-vowel.

TC-L-003 [MAJOR  ] Pada matra count ~14-16 (v2 updated)
  Criterion : Each pada approximately 14-16 matras (Tulsidas meter).

TC-L-004 [MINOR  ] Stock-word frequency cap (v2 recalculated)
  Criterion : Per-chapter caps:
                महान    <= 4
                अभिराम  <= 3
                जय-जयकार <= 3
                धाम     <= 5
                नाम     <= 5
                गान     <= 5

TC-L-005 [MAJOR  ] No banned filler words
  Criterion : Listed banned filler/coined words absent.

TC-L-006 [MINOR  ] Concrete imagery preferred
  Criterion : Subjective abstract-cliche avoidance.

TC-L-007 [MINOR  ] Register consistency within chaupai
  Criterion : No mixing of folk-Hindi and heavy Sanskrit-tatsama.

TC-L-008 [BLOCKER] Sthayi is kaand's signature Tulsidas couplet (NEW v2)
  Criterion : Sthayi lines 1-2 match the kaand's signature couplet from
              Procedure_Manual.txt Section 3.6 verbatim.

TC-L-009 [MAJOR  ] AABB pair full-syllable tail match (NEW v2)
  Criterion : Padas in each rhyme-pair share their FINAL 2 syllables
              (not just final consonant).

TC-L-010 [MAJOR  ] Word-sense correctness
  Criterion : Words used with correct semantic sense.

TC-L-011 [MAJOR  ] Awadhi participles present (NEW v2)
  Criterion : At least 2 distinct Awadhi-style participles per chapter:
              कीन्हा, दीन्हा, लीन्हा, आयो, गयो, जायो, छायो, समायो, भयो,
              भा, पाई, सुहाई, बढ़ाई, रचाई, समाना, स्थाना, वीरा, पीरा,
              धीरा.

TC-L-012 [MAJOR  ] No nukta ज़
  Criterion : Character ज़ does NOT appear in lyrics.
